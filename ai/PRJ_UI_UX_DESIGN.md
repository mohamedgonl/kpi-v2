# UI/UX Requirements (Government Style)

## 1. Visual Style
- Màu sắc: Chủ đạo là Xanh Navy (#1d4ed8) và Trắng. Điểm xuyết màu Đỏ nhạt cho các thông báo quan trọng.
- Font: Roboto hoặc Inter (Rõ ràng, dễ đọc).
- Cảm giác: Chuyên nghiệp, gọn gàng, không dùng quá nhiều hiệu ứng màu mè.

## 2. Dashboard Layout
- Dashboard cá nhân: Widget hiển thị tổng điểm hiện tại, biểu đồ tiến độ so với mục tiêu tháng.
- Dashboard lãnh đạo: Bảng xếp hạng các phòng ban/vụ, biểu đồ so sánh giữa các kỳ.

## 3. Form nhập liệu
- Thiết kế tối giản, hỗ trợ phím tắt để cán bộ nhập nhanh.
- Validation chặt chẽ: Không để trống các trường bắt buộc, kiểm tra định dạng ngày tháng.
